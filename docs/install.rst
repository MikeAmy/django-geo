.. |geo| replace:: Django Geo
.. |pkg| replace:: geo



Installation
============

In order to install |geo| simply use ``pip``::

   pip install django-geo

or ``easy_install``::

   easy_install django-geo


Configuration
=============

After installation add |pkg| to ``INSTALLED_APPS``::

   INSTALLED_APPS = (
       # ...
       'geo',
   )
